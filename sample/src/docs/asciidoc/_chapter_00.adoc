== Polynomials

Roots of a quadratic polynomial of type

[mathx, width=175, height=120]
----
$a x^2 + bx + c$
----

Are given by the formula:

[mathx, width=200, height=120]
----
\[ \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]
----
